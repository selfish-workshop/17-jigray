# MVP 범위 & 우선순위 — StayKo

## MVP (필수 — v1.0)

### P0: 핵심 (없으면 서비스 불가)

| # | 기능 | 설명 | 대상 |
|---|------|------|------|
| 1 | 매물 검색 & 필터 | 대학교·예산·기간으로 매물 검색 | 게스트 |
| 2 | 매물 상세 페이지 | 사진, 가격, 위치, 편의시설 정보 | 게스트 |
| 3 | 예약 신청 | 희망 날짜 + 메시지로 예약 신청 | 게스트 |
| 4 | 매물 등록 | 사진·가격·정보 입력하여 매물 게시 | 호스트 |
| 5 | 예약 승인/거절 | 게스트 프로필 확인 후 응답 | 호스트 |
| 6 | 다국어 UI | 영어 기본, 한국어 전환 가능 | 전체 |

### P1: 중요 (MVP에 포함하되 최소 기능)

| # | 기능 | 설명 | 대상 |
|---|------|------|------|
| 7 | 호스트-게스트 메시지 | 인앱 채팅 (자동번역 포함) | 전체 |
| 8 | 정산 대시보드 | 호스트별 수익·수수료·정산 현황 | 운영진 |
| 9 | 정산 실행 | 선택한 호스트에게 정산 처리 | 운영진 |
| 10 | 호스트 수익 현황 | 월별 수익·정산 완료 내역 확인 | 호스트 |

## 기술 스택 (데모 기준)

| 영역 | 기술 |
|------|------|
| Frontend | React + Next.js |
| Backend | Node.js + Express |
| DB | PostgreSQL |
| 인증 | OAuth 2.0 (Google, 카카오) |
| 파일 저장 | AWS S3 |
| 배포 | Vercel (FE) + Railway (BE) |
| 다국어 | i18next |

## 성공 지표 (데모 기준)

| 지표 | 목표 |
|------|------|
| 매물 등록 수 | 20개 이상 (데모 데이터) |
| 게스트 예약 전환율 | 검색 → 예약 10% 이상 |
| 호스트 응답 시간 | 평균 24시간 이내 |
| 정산 처리 시간 | 실행 후 1영업일 이내 |
