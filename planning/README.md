## 핵심 요약

### 서비스 컨셉
외국인 유학생이 한국에서 장기 숙소를 쉽게 찾고 계약할 수 있는 중개 플랫폼.
호스트(집주인)에게는 외국인 세입자 관리 부담을 줄여주고, 운영진이 정산을 대행한다.

### 3가지 사용자 유형
1. **게스트** (외국인 유학생) — 매물 검색·예약·계약
2. **호스트** (집주인) — 매물 등록·예약 관리·수익 확인
3. **운영진** — 수수료 정산·분쟁 중재·품질 관리

### MVP 흐름
```
게스트: 매물 검색 → 상세 확인 → 예약 신청
호스트: 예약 접수 → 게스트 확인 → 승인/거절
운영진: 정산 확인 → 수수료 계산 → 정산 실행
```

> 이 폴더는 기획 파트 리더가 상세화한 문서입니다.
