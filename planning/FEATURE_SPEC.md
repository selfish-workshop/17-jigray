# 기능 명세서 — StayKo

## 1. 게스트(외국인 유학생) 기능

### 1.1 매물 검색 & 필터
- **지역 필터**: 대학교 이름으로 검색 → 반경 2km 이내 매물 자동 표시
- **예산 필터**: 월세 범위(30만~100만원), 보증금 범위 설정
- **기간 필터**: 입주 희망일 ~ 퇴거 예정일 (최소 1개월)
- **매물 유형**: 원룸 / 투룸 / 쉐어하우스 / 오피스텔
- **편의시설 필터**: 가구 포함, Wi-Fi, 세탁기, 에어컨, 엘리베이터 등
- **정렬**: 추천순(기본) / 가격 낮은순 / 거리 가까운순 / 최신순

### 1.2 매물 상세 페이지
- 사진 갤러리 (최소 5장, 360도 뷰 권장)
- 매물 정보: 면적, 층수, 방향, 입주 가능일, 최소 계약 기간
- 비용 상세: 월세, 보증금, 관리비, 공과금 포함 여부
- 위치 지도: 가장 가까운 대학교·지하철역·편의점 표시
- 호스트 프로필 & 평점
- 후기 (이전 게스트 리뷰)
- **다국어 자동 번역** (원문 + 번역본 토글)

### 1.3 예약 신청
- 희망 입주일·퇴거일 선택
- 자기소개 메시지 (호스트에게 전달)
- 여권/비자 정보 업로드 (본인 확인용)
- 예약 상태 추적: 신청 → 호스트 검토 중 → 승인/거절

### 1.4 계약 & 결제
- 온라인 계약서 서명 (전자서명)
- 보증금 결제 (카드, 해외 송금, PayPal 등)
- 월세 자동이체 설정
- 계약서 PDF 다운로드

### 1.5 마이페이지
- 현재 계약 정보
- 결제 내역
- 찜한 매물 목록
- 호스트와 메시지

## 2. 호스트(집주인) 기능

### 2.1 매물 등록
- 기본 정보 입력: 주소, 면적, 층수, 방 수, 유형
- 사진 업로드 (최소 5장 필수)
- 비용 설정: 월세, 보증금, 관리비
- 편의시설 체크리스트
- 입주 가능 날짜 캘린더
- 매물 설명 작성 (한국어 → 영어 자동번역 제공)

### 2.2 예약 관리
- 예약 신청 목록 확인
- 게스트 프로필 확인 (학교, 비자 상태 등)
- 승인 / 거절 (거절 시 사유 입력)
- 계약서 발송 & 서명 확인

### 2.3 수익 대시보드
- 월별 수익 현황 그래프
- 정산 예정 금액
- 정산 완료 내역
- 매물별 수익 비교

### 2.4 호스트 프로필
- 소개, 응답률, 평균 응답 시간
- 게스트 후기 모아보기
- 본인인증 배지 (신분증 확인 완료)

## 3. 운영진 정산 페이지

### 3.1 정산 대시보드
- 전체 정산 현황 요약 (이번 달 총 정산액, 수수료 수익)
- 호스트별 정산 목록 (테이블)
  - 호스트 이름, 매물 수, 이번 달 수익, 수수료(10%), 정산 예정액
  - 정산 상태: 대기 / 처리중 / 완료
- 날짜 범위 필터

### 3.2 정산 실행
- 호스트 선택 → 정산 금액 확인 → 정산 실행 버튼
- 계좌 정보 확인 (은행, 계좌번호, 예금주)
- 정산 완료 시 호스트에게 자동 알림 발송

### 3.3 정산 이력
- 전체 정산 히스토리 (CSV 다운로드)
- 호스트별 누적 정산 금액
- 월별 수수료 수익 리포트

### 3.4 운영 관리
- 매물 신고 접수 & 처리
- 게스트/호스트 분쟁 중재
- 매물 품질 검수 (사진 대비 실제 확인)
