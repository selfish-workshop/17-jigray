# 사용자 시나리오 — StayKo

## 시나리오 1: 게스트 — "방 구하는 유학생 Tom"

### Who
- Tom, 미국 출신, 23세, 한국대학교 교환학생
- 한국어 거의 못함, 영어로 소통
- 한국 도착 2주 전, 아직 숙소 미정

### When
- 한국 입국 2~4주 전에 매물 검색 시작
- 입국 후 1주 이내 입주 희망

### Where
- 스마트폰 & 노트북 (해외에서 접속)
- 한국대학교 근처 원룸 희망

### Why
- 한국 부동산 앱은 전부 한국어 → 이해 불가
- 보증금·월세·관리비 구조를 모름
- 직접 부동산 방문은 언어 장벽으로 어려움

### How (사용 흐름)
1. StayKo 접속 → 영어 자동 설정
2. "한국대학교" 검색 → 근처 매물 리스트 표시
3. 예산 50만원, 원룸, 가구포함 필터 적용
4. 매물 3개 비교 → 상세 페이지에서 사진·위치·후기 확인
5. 마음에 드는 매물에 예약 신청 + 자기소개 작성
6. 호스트 승인 → 온라인 계약서 서명 → 보증금 PayPal 결제
7. 입주일에 열쇠 수령 → 생활 시작

## 시나리오 2: 호스트 — "외국인 세입자를 받고 싶은 김사장님"

### Who
- 김사장님, 55세, 대학가 원룸 3채 보유
- 한국어만 가능, 외국인 세입자 소통 부담
- 공실 기간을 줄이고 싶음

### When
- 공실 발생 시 매물 등록
- 예약 신청이 오면 당일~3일 내 응답

### Why
- 외국인 수요가 있지만 소통이 어려워 기피
- 기존 부동산 앱에서는 외국인 연결이 안 됨
- StayKo가 번역·계약·정산을 대행해주니 부담 감소

### How (사용 흐름)
1. StayKo 호스트 가입 → 본인인증
2. 매물 등록: 사진 촬영·업로드, 가격 설정 (한국어로 작성 → 영어 자동 번역)
3. 게스트 예약 신청 수신 → 프로필·비자 확인
4. 승인 → 계약서 자동 생성 → 게스트 서명 확인
5. 입주 후 매월 월세 자동 수금
6. 운영진이 수수료 제외 후 정산 → 계좌 입금

## 시나리오 3: 운영진 — "정산 담당자 박매니저"

### Who
- 박매니저, StayKo 운영팀, 정산 담당
- 매월 1일, 15일 정산 처리

### How (사용 흐름)
1. 정산 대시보드 접속 → 이번 기간 정산 대상 호스트 목록 확인
2. 호스트별 월세 수금 내역 확인 → 수수료(10%) 자동 계산
3. 정산 금액 확인 → "정산 실행" 클릭
4. 호스트 계좌로 이체 → 완료 알림 자동 발송
5. 정산 이력 CSV 다운로드 → 회계팀 전달
