# StayKo — 외국인 유학생 장기 임대 플랫폼

외국인 유학생을 위한 한국 장기 임대(원룸·쉐어하우스) 중개 플랫폼 데모. 호스트와 게스트를 연결하고, 운영진이 정산까지 관리한다.

## 프로젝트 정보
- **유형**: 웹서비스 데모 (부동산 중개 / 플랫폼)
- **타겟 사용자**: 게스트(외국인 유학생) + 호스트(집주인) + 운영진(정산 관리)
- **출시 목표**: 워크숍 발표 시연용 프로토타입

## 핵심 기능
- **게스트 모드**: 매물 검색·필터(대학 근처, 예산, 기간), 예약 신청, 계약 관리
- **호스트 모드**: 매물 등록·관리, 예약 승인/거절, 수익 현황 확인
- **운영진 정산 페이지**: 호스트별 정산 내역, 수수료 계산, 정산 실행·이력 관리
- 다국어 지원 (영어, 중국어, 일본어 등)
- 대학교 기반 지역 필터
